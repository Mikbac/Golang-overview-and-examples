# Basics

## Compile and install packages and dependencies

```shell
go install
```

## Add dependencies to current module and install them

```shell
go get
```

## Compile

```shell
go build <app>
# e.g.
go build main.go
# and run
./main
# or
main.exe
```

## Compile and execute

```shell
go run <app>
# e.g.
go run main.go
```

## Test packages

```shell
go test 
```

## Reforma package sources

```shell
go fmt
```
